Description

During the summer term of year 2, I worked in a group of 6 on an autonomous rover project. My main responsibility was to work on the ESP32 control code which integrated different subsystems together and was used to control the rover. I first had to establish communication between FPGA and ESP32 which used the UART protocol to transmit vision information from the camera and then Wi-Fi communication between the ESP32 and the webserver to send the telemetry from the rover to the server. I developed the main layout of the web application used to present the telemetry from the rover and I also implemented the image masking component in SystemVerilog for the FPGA. Due to our hard work and time spent on the project, we received the Head of Department’s Prize for the Best First Year Group Project.
