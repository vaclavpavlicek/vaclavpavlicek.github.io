Human voice synthesiser

Tools/Technologies

  • MATLAB/Simulink
  • C
  • Git

Description

Within the group of two, we developed a human voice synthesiser in Simulink and MATLAB as part of the module Real-Time Digital Signal Processing. The synthesiser was based on Linear Predictive Coding (LPC) - a method used during audio transmission (mobile phones or audio conferencing applications). We used Simulink Coder to generate a C code and upload it to the microprocessor TI2000.

Made by Václav Pavlíček (vaclav@pavlicek.cz).
