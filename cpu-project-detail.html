1st-year CPU Project

Our team won the Head of Department’s Prize for the Best First Year Group Project. The project involved creating a CPU in a graphical editor ISSIE. I pipelined the CPU and then I also implemented two versions of the floating point unit (FPU) - one version optimized for latency and the other for throughput.

Made by Václav Pavlíček (vaclav@pavlicek.cz).
