Description

As part of the Instruction Architecture and Compilers module, our task was to write C-to-MIPS Compiler in a group of two. The compiler was written in C++ with the use of Bison and Yacc. Our compiler received the 5th highest score out of approximately 80 compilers and its source code can be found on GitHub.
